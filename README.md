⭐ Machine Learning Project | K-Means & Hierarchical Clustering

## Problem Statement
HELP International NGO wants to identify countries that require financial aid.

## Dataset
167 countries with socio-economic indicators:
- GDP per capita
- Income
- Child mortality
- Life expectancy
- Inflation

## Methodology
1. Exploratory Data Analysis
2. Feature Scaling
3. Elbow Method
4. K-Means Clustering
5. Hierarchical Clustering

## Results
Countries were grouped into:
- Developed
- Developing
- Underdeveloped

## Countries Recommended for Aid
- Burundi
- Liberia
- Congo (Democratic Republic)
- Niger
- Sierra Leone

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
